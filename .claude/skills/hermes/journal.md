# Hermes journal

Hermes appends one entry per run. Read this file at the start of every run and
apply prior lessons before choosing fast-path vs. browser-fallback.

Format:

```
## YYYY-MM-DD HH:MM <one-line task>
- host: <example.com> — fast path: ok | failed (<reason>)
- lesson: <what to do differently next time>
```
